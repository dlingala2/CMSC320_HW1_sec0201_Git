Answer: We might consider using Git because it helps track changes to our code, making it easier to collaborate with others and go back to earlier versions if needed.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging combines two branches by creating a new commit that keeps the full history of both. Rebasing however moves or rewrites commits from one branch on top of another, creating a cleaner and more straightforward history for the commits.
